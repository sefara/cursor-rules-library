# cursor-rules-library

A public library of `.cursor/rules` templates for AI-agent-driven development with Cursor 2.0.

This project is a personal collection of officially recommended and AI-generated `.mdc` rules that I use in my projects. The library is continuously updated and refined over time.

## Features

- Ready-to-use `.cursor/rules` templates for Cursor 2.0.
- Mix of officially recommended and AI-generated rules.
- Ocasionally updated and maintained.

## Usage

To use these rules in your project:
